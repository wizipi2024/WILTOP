=== TROUBLESHOOTING ===

PROBLEMA 1: Nao aparece [ACAO] amarela
CAUSA: Sistema nao autorizado
SOLUCAO:
  1. Execute: AUTORIZAR_TUDO.py
  2. Digite: SIM AUTORIZO TUDO
  3. Abra novamente: ABRIR_COMPLETO.bat

PROBLEMA 2: Erro ao abrir
CAUSA: Dependencias faltando ou ambiente nao ativado
SOLUCAO:
  1. Execute: DIAGNOSTICO.bat
  2. Veja o que esta faltando
  3. Se precisar: venv\Scripts\activate
  4. pip install -r requirements.txt

PROBLEMA 3: Nao entende comandos
CAUSA: Executor inteligente nao esta ativo
SOLUCAO:
  1. Verifique autorizacao (deve estar em VERDE)
  2. Tente comandos simples primeiro: "abra o bloco de notas"
  3. Seja especifico: "crie um arquivo teste.txt na area de trabalho"

PROBLEMA 4: API Error
CAUSA: Problema com Groq API
SOLUCAO:
  1. Verifique .env
  2. Confirme que GROQ_API_KEY esta la
  3. Teste a conexao
